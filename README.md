# Dino Game (C++ & Raylib)

A simple clone of the classic Chrome Dino game built using **C++** and the **Raylib** library.

## Features

* Basic jump and obstacle mechanics
* Score tracking
* Simple collision detection

## Controls

* **Space** – Jump
* **Esc** – Exit game

## Requirements

* C++ compiler (e.g., g++, clang++)
* Raylib installed

## Notes

This project was created as a learning exercise for working with Raylib and basic game development concepts in C++.
